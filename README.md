## Youtube Chat Thing

Redirect to YouTube Live Chat using only your channel handle. No video ID required.

Use: https://yt-chat-thing.cjmaxik.com/@YourChannelHandle \
(e.g., https://yt-chat-thing.cjmaxik.com/@LofiGirl)
